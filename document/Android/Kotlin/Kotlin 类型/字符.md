# 字符

字符用 `Char` 类型表示。 字符字面值用单引号括起来: `'1'`。

特殊字符可以以转义反斜杠 `\` 开始。 支持这几个转义序列：

* `\t`------制表符
* `\b`------退格符
* `\n`------换行（LF）
* `\r`------回车（CR）
* `\'`------单引号
* `\"`------双引号
* `\\`------反斜杠
* `\$`------美元符

编码其他字符要用 Unicode 转义序列语法：`'\uFF00'`。

```kotlin
fun main() {
//sampleStart
    val aChar: Char = 'a'

    println(aChar)
    println('\n') // 输出一个额外的换行符
    println('\uFF00')
//sampleEnd
}
```

如果字符变量的值是数字，那么可以使用 `digitToInt()` 函数将其显式转换为 `Int` 数字。
